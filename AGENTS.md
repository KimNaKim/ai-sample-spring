# Agents & Skills

이 프로젝트의 작업을 수행하는 전문 에이전트 및 스킬 목록입니다.

| 분류 | 이름 | 트리거/용도 | 주요 능력 |
| :--- | :--- | :--- | :--- |
| **Agent** | `git-commit-agent` | `커밋해줘`, `커밋 메시지 작성해줘`, `작업 내역 정리해서 커밋해줘` | 변경된 파일과 코드 수정 내역을 분석하여 상세한 커밋 메시지를 자동 생성하고 커밋을 수행하는 전문 에이전트. |
| **Skill** | `compound` | `/compound`, `컴파운드 실행`, `컴파운드 모드` | '선 계획, 후 승인, 사후 보고' 프로세스를 강제하는 워크플로우. |
| **Skill** | `deep-talk` | `/deep-talk`, `깊은 대화`, `같이 생각해보자` | 깊이 있는 대화/토론/질의응답이 필요할 때 사용하며, 코드 수정 없이 설계를 논의함. |
| **Skill** | `deepinit` | `/deepinit`, `코드베이스 문서화해줘`, `AI-CONTEXT.md 생성해줘` | 코드베이스를 인덱싱하고 계층형 AI-CONTEXT.md 파일을 생성함. |
| **Skill** | `frontend-design` | UI 설계 요청, Mustache 템플릿 생성 요청 | 최신 웹 트렌드를 반영하여 UI를 설계하고, 디자인 토큰을 활용해 SSR용 Mustache 템플릿을 생성함. |
| **Skill** | `plan` | `플랜 실행해줘` | 코드 작성 전 '선 계획, 후 승인, 사후 보고' 프로세스를 강제함. |
| **Skill** | `report` | `/report`, `보고서 작성해줘`, `작업 결과 정리해` | 방금 완료한 작업의 전체 흐름을 구조화된 보고서로 작성함. |
| **Skill** | `why` | `/why`, `왜 이런 일이?`, `원인 분석해줘`, `규칙 갱신해줘` | 실수 교정(원인 분석) 및 구현 완료 후 새로운 규칙을 규칙 파일에 반영함. |
